CSS backdrop-filter

6 Oct 2023

The backdrop-filter property lets you apply graphical effects to the area behind an element.

backdrop-filter - CSS: Cascading Style Sheets | MDN The backdrop-filter CSS property lets you apply graphical effects such as blurring or color shifting to the area behind an element. Because it applies to everything behind the element, to see the effect you must make the element or its background at least partially transparent.
